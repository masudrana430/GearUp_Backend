# GearUp Backend API

GearUp is a backend API for renting sports and outdoor equipment instantly.

Customers can browse available gear, create rental orders, make payments and submit reviews. Providers can manage inventory and fulfil rental orders. Admins can manage users, categories, listings, rentals and payments.
